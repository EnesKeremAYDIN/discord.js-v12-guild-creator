# Discord.js v12 Guild Creator

A Node.js bot using Discord.js v12 to create Discord servers automatically. This script is ideal for bots with fewer than 10 servers, as they can only create servers up to a total of 10. For example, a bot in 0 servers can create 10 servers, but if itâ€™s in 3 servers, it can create only 7 more.

## Setup and Usage

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/EnesKeremAYDIN/discord.js-v12-guild-creator.git
   cd discord.js-v12-guild-creator
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Configure and Run**:
   - Add your Discord bot token in `app.js`.
   - Adjust server setup details as desired.
   ```bash
   node app.js
   ```
